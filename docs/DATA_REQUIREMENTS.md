# Data Requirements for GeoAI Agent

To replace the "Mock Data" with real-world data, please locate files meeting the following specifications:

## 1. Flood Hazard Maps 🌊
**Purpose:** To calculate the flood risk score for a specific location.
*   **Preferred Format:** `GeoTIFF` (`.tif`)
    *   This is a "Raster" format where every pixel represents a value.
*   **Alternative Format:** `ESRI Shapefile` (`.shp`) or `GeoJSON`.
*   **Content:**
    *   Pixel Value should represent **Water Depth** (meters) or **Hazard Level** (1=Low, 2=Medium, 3=High).
    *   Should verify the Return Period (e.g., "50-year return period" or "100-year return period").
*   **Area:** Must cover **Colombo North** (Kelani River mouth, Mattakkuliya, Modara).
*   **Source Suggestions:**
    *   RiskInfo Sri Lanka (http://www.riskinfo.lk/)
    *   Global Flood Awareness System (GloFAS)
    *   NBRO (National Building Research Organisation) - Digital datasets if available.

## 2. UDA Zoning Plans 🏗️
**Purpose:** To verify land-use regulations (RAG Knowledge Base).
*   **Format:** `PDF` (Text-selectable preferred, not scanned images).
*   **Specific Document:** "Colombo Commercial City Development Plan 2019-2030".
*   **Key Pages:** Zoning definitions for "Special Primary Residential Zone" etc.
